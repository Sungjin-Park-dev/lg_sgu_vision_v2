### Isaac Sim 환경 정리 — 캐시 권한 + OpenWindow 에러

**캐시 권한 복구**: `.venv/.../isaacsim/kit/cache/DerivedDataCache/` 의 `largecachedata_{1,2,3}` 가 `root:root` 로 박혀 있어 `sungjin` 으로 실행 시 LocalDataStore append 실패 → 후속으로 `SharedMutex does not allow recursive locks` 어서션까지 연쇄. 컨테이너 안에서 root 로 실행했던 흔적. `chown sungjin:sungjin` 으로 해결.

### 검사 카메라 prim 추가 — config.py 스펙으로

**위치 결정**: 처음엔 `tool0`/`wrist_3_link` 자식으로 두고 SG8S world transform 을 카피했으나 시각적으로 어긋남.
→ **SG8S Xform 자식** (`/World/UR20/.../wrist_3_link/SG8S_AR0820C_5300_G2A_H30YA/InspectionCamera`) 으로 변경. SG8S subtree 안의 기존 Camera prim 을 탐색해 그 world transform 을 SG8S 기준 local 로 환산해 박음 — 기존 카메라와 동일 위치/방향에 겹침.

**intrinsics 매핑** (UsdGeom.Camera, 단위 mm):
| USD attr | 값 | 근거 |
|---|---|---|
| `focalLength` | `CAMERA_WORKING_DISTANCE_MM` (110) | 핀홀 모델에서 `FOV(@d) = d · aperture / focalLength` 가 작업거리에서 정확히 맞음 |
| `horizontalAperture` | `CAMERA_FOV_WIDTH_MM` (41) | 동일 |
| `verticalAperture` | `CAMERA_FOV_HEIGHT_MM` (30) | 동일 |
| `focusDistance` | WD × 1e-3 (= 0.11 m) | stage 단위(m) |
| `clippingRange` | (0.01, 5.0) m | |

DOF 관련 `fStop` 은 미설정 — 검사 시 초점 흐림 시각화가 필요해지면 추가.

### ROS2 publisher — Action Graph 통합

처음엔 카메라용 `/CameraGraph` 를 따로 만들었으나, joint state 그래프와 동일하게 `OnPlaybackTick` + `ROS2Context` 만 공유하면 되므로 **단일 `/ActionGraph` 안에 통합**. 카메라 prim 이 잡혀야 노드를 추가하도록 가드.

추가 노드:
- `IsaacCreateRenderProduct` (cameraPrim = InspectionCamera, size = publish 해상도)
- `ROS2CameraHelper × 2` — RGB / Depth
- `ROS2CameraInfoHelper`

토픽:
- `/inspection_camera/image_raw`
- `/inspection_camera/depth`
- `/inspection_camera/camera_info`
- frame_id `inspection_camera`

### 해상도 — 풀해상도 발행 시 렉

4096 × 3000 = 12.3 MP × ~30 FPS = 1.5 GB/s 수준이라 GUI 가 멈춤. 별도 publish 해상도 필드 도입:
- `CAMERA_RESOLUTION_W/H = 4096/3000` — 센서 원본(다른 계산 시 진실값)
- `CAMERA_PUBLISH_W/H = 1024/750` — Isaac Sim 렌더/ROS2 발행용 (4× 다운샘플, 비율 유지)

추후 검사 알고리즘 검증에 풀해상도가 필요해지면 발행 해상도만 올리면 됨.

## 결정/변경

- **셸 환경 통일**: 모든 실행은 `ros-jazzy` 컨테이너 안. 호스트 venv 활성화 금지. 컨테이너 안에서도 root 셸로 실행 금지 (cache/USD 권한 오염).
- **InspectionCamera 부모 = SG8S Xform**: SG8S 내부 좌표계와 일관성. SG8S 가 wrist_3_link 자식이라 EE 체인은 자동 추종.
- **카메라 transform 은 SG8S 내부 기존 Camera 그대로 복사**: payload 가 정의한 광학 중심 / 회전을 단일 진실원으로 둠. payload 가 바뀌어도 자동 추종.
- **단일 ActionGraph**: 별도 그래프는 evaluator 가 매 틱마다 따로 돌아 부하 증가. 통합 후 OnPlaybackTick + ROS2Context 공유.
- **`CAMERA_PUBLISH_W/H` 분리**: 센서 스펙 ≠ 발행 해상도. 진실원/런타임 부하 분리.
